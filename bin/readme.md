

# airrohr-firmware

# Instruction for update 
* Erase Flash (with Airrohr Flasher or FlashESP8266 see below)
* Run (on Windows) FlashESP8266
* Select com port
* Select blank_4MB.bin to erase
* Select firmwarexx.bin
